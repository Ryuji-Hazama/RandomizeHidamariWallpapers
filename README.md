# Randomize the Live Wallpaper at the startup

You can automatically randomize the Hidamari live wallpaper at the startup.

## Usage

1. Run `pip install maplex`
2. Set path for `randomAndRun.sh` in `Hidamari.desktop`
3. Place `Hidamari.desktop` to `/home/username/.config/autostart/`
4. Remove `io.github.jeffshee.Hidamari.desktop` from `/home/username/.config/autostart/`
